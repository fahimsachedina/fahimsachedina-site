# fahimsachedina.com

Personal site for Fahim Sachedina — built to be fast, free to host, and actually stay up.

## What it is

A clean, single-page personal site covering who Fahim is, what he's building, and where to find him. Built with plain HTML/CSS, deployed on Vercel, domain managed via WordPress.com.

## Goals

- Be a simple, reliable online presence that doesn't go down
- Point people to the right places — Work in Crypto, Optimism, Beta Studios, Azamour Capital
- Load fast, look good, cost nothing to run

## Stack

- **HTML/CSS** — no frameworks, no build step
- **Vercel** — free hosting, auto-deploys on every push to `main`
- **GitHub** — source of truth for all code changes
- **Domain** — fahimsachedina.com registered via WordPress.com, DNS pointed to Vercel

## Sections

- **Hero** — name, tags (GTM / Partnerships / Tech / Crypto / AI), CTA to Work in Crypto
- **Work** — Optimism, Work in Crypto, Beta Studios, Azamour Capital
- **Featured** — Work in Crypto callout
- **About** — short bio and stat cards
- **Contact** — X (@fahimsach), GitHub (@fahimsachedina), LinkedIn (/fahimsach)
